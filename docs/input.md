`input` (component)
===================

Wrap a Native Base input with a label and eventualy an icon by is name.
Provide the onChange to get the text entered
Flag success or error to display corresponding state

Props
-----

### `error`

type: `bool`
defaultValue: `false`


### `iconName`

type: `string`
defaultValue: `null`


### `label` (required)

type: `string`


### `onChange` (required)

type: `func`


### `success`

type: `bool`
defaultValue: `false`
