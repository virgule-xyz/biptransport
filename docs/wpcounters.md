Affiche une ligne avec les compteurs d'objectifs de livraison et de dépôt

type: `string`

### `colorShip`

type: `string`

### `frontColorPick`

type: `string`

### `frontColorShip`

type: `string`
defaultValue: `'#fff'`


### `pickup` (required)

type: `union(number|string)`


### `shipping` (required)

type: `union(number|string)`
